# JanSevaAI 🚀

Telugu Voice-Based AI Assistant for Government Schemes

## Features
- 🎤 Telugu Speech-to-Text (Whisper)
- 🤖 Gemini AI Responses
- 🔊 Telugu Text-to-Speech
- 📚 RAG-based Scheme Retrieval

## Tech Stack
- Python
- OpenAI Whisper
- Google Gemini API
- gTTS
- FAISS (RAG)

## How to Run

1. Create virtual environment
2. Install dependencies:
   pip install -r requirements.txt

3. Run:
   python stt.py
